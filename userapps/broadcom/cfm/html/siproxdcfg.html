SIP ALG - Configuration

Session Intitation Protocol Application Level Gateway (SIP ALG) is a daemon for the SIP protocol. It handles registrations of SIP clients on a private IP network and performs rewriting of the SIP message bodies to make SIP connections possible via an masquerading firewall. It allows SIP clients (like kphone, linphone) to work behind the DSL Router.

Select the desired values and click "Save/Apply" to enable or disable SIP proxy.

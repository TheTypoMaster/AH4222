Journal Système - Configuration
 
- Si le mode "Journal" est activé, le système enregistrera tous les événements sélectionnés.
- Pour le niveau de "Journal", tous les événements au dessus ou égaux au niveau sélectionné seront sauvegardés.
- Pour le niveau d'affichage, tous les événements au dessus ou égaux au niveau sélectionné seront affichés.
- Si le mode sélectionné est " Distant "ou "Les deux", les événements seront envoyés à l'adresse IP spécifiée et au port UDP du serveur du système " journal ".
- Si le mode sélectionné est " local" ou "Les deux", les événements seront enregistrés sur la mémoire local.
 
Sélectionner les valeurs souhaitées et cliquer sur " Sauvegarder/Appliquer" pour configurer les options du système « Journal ».

NAT - Port Triggering
 
Certaines applications (Jeux, vidéo conférence…) nécessitent que des ports spécifiques soient ouverts dans le firewall du modem pour en permettre l’utilisation.
 
Pour la configuration, sélectionner les applications préenregistrées ou créer votre propre application en remplissant les champs ci-dessous.
 
Cliquer sur "Sauver/Appliquer" pour appliquer les modifications.
 
Rappel: Vous pouvez configurer un maximum de <%ejGet(glbEntryCt)%> règles.

Tester la connexion USB
 
Succès Indique que le port USB de votre ordinateur est connecté au port correspondant sur le modem. Un voyant vert sur le modem indique aussi que la connexion est établie et que le test est réussi. Il clignote lors du transfert de données.
Echec Indique que le modem ne détecte pas le port USB de votre ordinateur.
Si le test échoue, suivez la procédure de dépannage listée ci-dessous et ré-exécutez les tests de diagnostics en cliquant sur le bouton en bas de page.
Si tous les tests réussissent, fermez et redémarrez votre navigateur Web pour accéder à Internet.
 
Dépannage:
 
1. Si le port USB de votre modem n’est pas connecté, ce test devrait échouer.
 
2. Si vous ne pouvez pas accéder à cette page, Vérifiez que votre câble USB de votre ordinateur (ou Hub) est bien connecté au port USB du modem. Testez de nouveau le câble en débranchant ses deux extrémités avant de les rebrancher sur leur port respectif.
 
3. Eteignez votre modem, attendez 10 secondes avant de le rallumer.
 
4. Une fois le modem allumé, appuyez pendant au moins cinq secondes sur le bouton "RESET" puis relâcher le. Cette procédure permet de réinitialiser les paramètres d'usine. Attendez que le modem s'initialise, puis fermez et ré-ouvrez votre navigateur Internet.
Pour le reconfigurer, tapez votre nom d'utilisateur et votre mot de passe fournis par Club Internet.
 
Si vous avez suivi toute la procédure de dépannage ci-dessus et qu'elle n'a pas abouti, contacter la Hotline de Club Internet.
